# Bug Tracking System (Java Swing)

A desktop-based Bug Tracking System built using **Java Swing**, designed to manage software bugs efficiently by allowing admins, developers, and testers to track, update, and resolve issues within a project workflow.

---

## 📌 Features
- **User Authentication**
  - Admin login
  - Developer/Testers login
- **Bug Management**
  - Add new bugs
  - Assign bugs to developers
  - Update bug status (Open, In-Progress, Fixed, Closed)
  - Delete or search bugs
- **User Management (Admin Only)**
  - Add/Edit/Delete developers or testers
- **Dashboard**
  - Summary of open, closed, and pending bugs
  - Developer-specific bug list
- **Search & Filter**
  - Search bugs by ID, title, priority, or status
- **Persistent Storage**
  - Text files / JSON / Database (depending on your implementation)

---

## 🛠️ Technologies Used
- **Java 17+**
- **Swing UI**
- **OOP + MVC Architecture**
- **File-based storage or MySQL** (optional)
- **IntelliJ IDEA / NetBeans / Eclipse**
